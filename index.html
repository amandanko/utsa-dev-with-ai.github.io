Tool layer — the playbooks

Concept layer

The eight modules

Each module is 15–25 minutes of self-paced video and reading plus a short hands-on exercise. Finish the exercise before your class Q&A session — the discussion is built on what you bring.

01

The Landscape & Your Stance

Autocomplete, agents, and autonomous systems — what each working mode is for, what "vibe coding" actually costs you, and the integrity rules for this course.

Durable~20 min
 02

Specification Before Generation

Design docs, prototypes, and prompts as specifications. If you can't state what done looks like, no agent can build it for you.

Durable~20 min
 03

Working with an Agent

Scoping tasks, managing context, iterating on results — and recognizing the moment to take the wheel back.

Durable~25 min
 04

Reading Code You Didn't Write

Code review as a first-class skill. Tracing behavior, spotting smells, and deciding whether generated code earns a place in your project.

Durable~25 min
 05

Verification & Testing

Tests as the contract between you and the agent. Nothing generated gets merged unverified — here's how to make that practical.

Durable~25 min
 06

Security & Failure Modes

Hallucinated packages, injected dependencies, license problems, and the review habits that catch them before they ship.

Durable~20 min
 07

Troubleshooting & Debugging with AI

Debugging your code with an agent — and debugging the agent's own output when it confidently goes wrong.

Durable~20 min
 08

Keeping Up

How to evaluate a brand-new tool in 30 minutes. You'll do it for real — and your write-up may become this site's next playbook.

Durable~20 min
